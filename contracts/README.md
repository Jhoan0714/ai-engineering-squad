# Contracts

Handoff artifact shapes used between roles (acceptance criteria, OpenSpec refs, risk notes, check mapping, signoff).

Designed to be **embeddable**: a future H0 `validate` entrypoint should be able to check these without rewriting role content.

Content lands in backlog **D3**. Prefer versionable schemas if programmatic validation is required (see [docs/open-questions.md](../docs/open-questions.md) #8).
