# Roles

Canonical role packs for the AI Engineering Squad.

Each role lives in `roles/<role-id>/AGENT.md`. See [docs/layout.md](../docs/layout.md) and [docs/roles.md](../docs/roles.md).

## MVP

| Role ID | Title |
|---------|-------|
| `product-manager` | Product Manager |
| `senior-software-engineer` | Senior Software Engineer |
| `qa-engineer` | QA Engineer |
| `automation-engineer` | Automation Engineer |

`AGENT.md` content for each role is delivered in backlog items **B1–B4** (separate PRs).
